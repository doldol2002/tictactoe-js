# Tic Tac Toe (Vanilla JavaScript)

A browser-based implementation of the classic Tic Tac Toe game built using HTML, CSS, and JavaScript.

## Overview

This project demonstrates DOM manipulation, state management, and win-condition evaluation without external libraries.

## Features

* Two-player gameplay
* Dynamic board rendering
* Win detection algorithm
* Draw detection
* Reset functionality
* Responsive layout

## Technical Design

Game state is stored in an array representing the board.
After each move, the program evaluates predefined winning combinations to determine the game outcome.

The architecture separates:

* Game state logic
* Win evaluation logic
* DOM rendering updates

## Tech Stack

* HTML5
* CSS3
* Vanilla JavaScript (ES6)

## Future Improvements

* Implement AI opponent using the Minimax algorithm
* Add persistent score tracking
* Add animation transitions
